# Phase 3 — Compound + measure (weeks 9–12)

> Owner copy. Source of truth for standups.

## Goal

Harden AI Search citation moats on the Phase 2 pages. Measure drift against the Phase 1.4 baseline. Quarterly retro.

## Work items

### 3.1 — Run /seo-geo on /vs/webflow, /vs/squarespace, /studio/agencies
- Owner: SEO Lead
- Effort: M (~half day per URL)
- Skill: seo-geo
- Source: the URLs shipped in Phase 2.2 and 2.4.
- Output: per-URL GEO recommendations and current AIO citation status.
- Phase-end metric: each new page has explicit GEO improvement actions identified; AIO citation status documented (cited vs not cited).

### 3.2 — Run /seo-backlink-gap vs webflow.com + squarespace.com
- Owner: Outreach Team
- Effort: L (skill run is fast; outreach execution takes the rest of Phase 3)
- Skill: seo-backlink-gap
- Output: prospect list — referring domains that link to webflow.com + squarespace.com but not wix.com.
- Phase-end metric: prospect list delivered + first 20 outreach emails sent.

### 3.3 — Run /seo-drift compare wix.com against Phase 1.4 baseline
- Owner: SEO Lead
- Effort: S
- Skill: seo-drift (compare subcommand)
- Cost: ~20–30 SE Ranking credits + 1 Firecrawl credit (per SKILL.md)
- Dependency: Phase 1.4 baseline must exist.
- Output: examples/seo-drift-wix-com-<date>/DRIFT-REPORT.md
- Phase-end metric: RED/YELLOW/GREEN deltas surfaced. "What to investigate first" recommendation captured.

### 3.4 — Quarterly retro
- Owner: SEO Lead
- Effort: M
- Inputs: pillar score deltas (4 scores: Technical/Content/Topical/AI Search), AI Search SoV delta vs 2026-04-27 baseline, Phase 2 pages ranking position, Phase 3.2 outreach results.
- Output: Q3 plan signed off by stakeholders.
- Phase-end metric: Q3 plan exists.

## Phase exit criteria

- Drift compare shows no RED-tier regressions.
- AI Search SoV held or grew on the cluster covered by the new Phase 2 pages.
- Outreach team executing on backlink-gap prospect list (≥20 outreach emails / week).
- Q3 plan signed off.

## Re-run cadence after Phase 3

- /seo-drift compare wix.com — monthly. Catches silent regressions.
- /seo-plan wix.com — quarterly. Re-anchors against latest drift compare.
- /seo-ai-search-share-of-voice — quarterly. Tracks SoV % across the 5 engines vs the 2026-04-27 baseline.
- /seo-technical-audit — quarterly OR after any major site redesign / deploy.
- /seo-content-audit — quarterly on the new top-10 pages list.

## Total estimated cost

SE Ranking: ~50 credits across the specialist runs.
Firecrawl: ~5 credits.
Outreach team time: 4 weeks active outreach.

## Critical-path notes

Phase 3.3 is the loop-closer. If the drift compare surfaces a RED-tier regression (DA dropped ≥10, traffic dropped ≥50%, top-3 keyword count dropped ≥40%), Q3 planning pivots to investigate that regression before continuing the Wix Studio agency push. Don't compound on a destabilising base.
