# evidence/

Raw HTML evidence for the per-URL hreflang inventory in `02-per-url-hreflang.md`.

The full `rawHtml` payloads for each sampled URL average 1.2–2 MB each — not committed to git for an example deliverable. Production runs persist them here under filenames like:

- `homepage-rawhtml.html`
- `sample-1-airbnb-co-uk-rawhtml.html`
- `sample-2-airbnb-es-rawhtml.html`
- `sample-3-airbnb-de-rawhtml.html`
- `sample-4-airbnb-fr-rawhtml.html`
- `sample-5-airbnb-com-help-rawhtml.html`
- `sitemap.xml` (if `seo-sitemap` is run as a successor)

For this example, the parsed evidence (canonical, hreflang counts, self-reference checks, x-default flags) is consolidated directly in `02-per-url-hreflang.md`.
